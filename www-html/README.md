# Hydrogen Planet Survey Landing Page

A cosmic-themed landing page for the Hydrogen Planet survey, built with HTML, CSS, Bootstrap 5, and JavaScript.

## Project Structure

```
hydrogen-planet-landing/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Custom styles
├── js/
│   └── main.js         # JavaScript functionality
├── assets/
│   └── images/         # Place your logo here
└── README.md           # This file
```

## Setup & Installation

1. **No build process required** - This is a static HTML project
2. Place your logo in `assets/images/` directory (e.g., `logo.png`)
3. Open `index.html` in any modern web browser

## Running the Project

Simply open `index.html` in your browser:
- Double-click the file, or
- Right-click → Open with → Your preferred browser

Alternatively, use a local development server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (npx)
npx serve

# Using PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.

## Customization

### Adding Your Logo
- Place your logo image in `assets/images/logo.png`
- Update the `src` attribute in `index.html` if using a different filename

### Adding Your Content
- Locate the content section in `index.html` (marked with comments)
- Paste your survey content in the designated area

### Customizing Colors
- Edit the CSS custom properties in `css/styles.css` to adjust the cosmic theme colors

## Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Custom styling and animations
- **Bootstrap 5.3** - Responsive layout and components
- **JavaScript** - Interactive features and animations

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge)
